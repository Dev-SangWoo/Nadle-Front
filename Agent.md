## 📌 프로젝트 개요

- **주제**: AI가 추천해주는 미션형 자전거 동네 여행
- **목표**
    - 사용자의 위치 기반으로 동네 자전거 여행 경로를 추천
    - 게임 요소(미션/스탬프)를 결합한 체험형 서비스 제공
- **기대 효과**
    - 공영자전거 이용 활성화를 통한 친환경 교통수단 확산 및 도시 내 탄소 배출 저감 기여
    - 일상 속 자연스러운 이동과 여가 결합을 통해 국민 건강 증진 및 생활 만족도 향상
    - 지역 관광자원과 연계한 자전거 이동 경로 제공으로 지역 경제 활성화 및 관광 소비 유도

## 🖥️ 서비스 구성

### 1. 메인 화면 (챗봇)

- 사용자가 원하는 컨셉의 루트를 텍스트로 입력 (ex. 벚꽃 볼 수 있는 30분짜리 코스 추천해줘)
- 사용자 현재 위치 + 입력 텍스트 + 주변 관광정보 데이터 기반 추천 관광지 리턴
- 관광지 이름 클릭 시 상세 정보 출력
- 원하는 루트 선택할때까지 사용자가 입력 가능

### 2. 대여소 출력 및 선택

- 현재 위치 기반 지도 (주변 자전거 대여소 아이콘 표시) + 대여소 리스트 표시
- 대여소 클릭 → 상세 정보 출력
- 대여소 선택 완료 버튼 클릭

### 3. 경로 출력

- 선택한 대여소(출발지) + 챗봇에서 선택한 AI 추천 관광지들로 지도에 루트 생성 (카카오맵)

### 4. 미션 요소

- 각 관광지 스팟 도착 시 미션 수행
    - 사진 인증
    - 관광지 관련 OX 퀴즈
    - 휴식 타이머 작동
- 미션 완료 시 스탬프

### 5. 최종 루트 종료

- 최종 보고서 출력
    - 시간, 거리, 코스, 관광지 정보, 스탬프 등

### 6. 반납 대여소 선택

- 최종 목적지 주변 대여소 지도에 출력

## 사용자 흐름 



상단 네비 : 로고만 
하단 네비 : 지도 페이지(가운데 크게) + 오른쪽 옆에 여행내역 

탭 : 지도 /  여행내역 

```

시작하기 -> 맞춤, 추천 프롬프트 작성 or 우리가 제공해주는 테마 선택
			-> 해당 테마나 프롬프트 기준으로 경로 추천 -> 맘에 안들면 다시 조정 가능 (ai 챗봇)
				-> 주변 대여소 정보 출력
					-> 해당 정보 누르면 상세 대여소 정보 출력(모달)
						-> 대여소 선택하면 해당 대여소 기반으로 경로 설정
							-> 전체 지도 경로 및 현재 위치 출력(이동중)
									-> 목적지 당 미션 1개 랜덤 생성 (AI) 
											-> 목적지 근처 도착하면(사용자가 도착완료) 미션 수행(모달) 
													-> 미션 완료하면 스탬프 증정
															-> 마실 결과지 출력
																-> 마실 종료 시 마지막 목적지 기준 주변 대여소 출력
```
